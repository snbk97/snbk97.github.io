The ls Guide

2019-06-28
#linux  #manpage  #ls 

You must be wondering why even bother reading a post about ls command out of all the commands.

  1. It’s hard to remember all the options, so here’s a cheat sheet.
  2. It’s a handy tool to search through a pile of files, so why not read more about it.
  3. Some awesome tools, I came across - shared at the end.

ls - this command is used for listing directory contents on *nix OS.

SYNTAX - ls [OPTION] [FILE]

Popular Options

  • ls - lists contents with almost no formatting
  • ls -l - formatted as a list; shows more info shown such as: permissions, date, time, author, owner etc.
  • ls -a - shows all files and folders available, including the hidden ones including (.) current dir as well as (..) parent dir
  • ls -lh - size of files are now more human readable
  • ls -lS - lists files sorted by file size in descending order
  • ls -ltr - lists files sorted by modification time (-r for reverse order).
  • ls -m - delimits file list with ‘,’ commas. (see –FORMAT flag for more info)
  • ls -R - Recursively lists files and folders. (you can pair -l with it too.)
  • ls *.mp3 - filter out any file extension similarly

Other Options

  • -A - lists all, excluding (.) current dir as well as (..) parent dir.
  • -d - lists directories and not their contents. eg: ls -d */
  • -r - for showing output in reverse order
  • -f - ls -a as unsorted and color mode disabled
  • -i - print inode values (index numbers) of files
  • -g - omits owner column from listing
  • -o - omits group column from listing
  • -n - user and group id’s displayed as numeric values.
  • -N - print file names without quotes
  • -X - sort listing by extension in ascending order
  • color - add color to output. values = ‘auto’, ‘none’ etc.
  • file-type - same as ls *.mp3
  • --full-time - shows time in full ISO formatting
  • --help - display this help and exit
  • --version - output version information and exit

You can read more about the ls command here.

Or run man ls on your terminal.

—–

Some awesome tools to consider:

  1. lsd - A color coded version of the ls command. Available as a snap package on the snapstore.

  2. cheat - lists usage for popular commands in *nix systems Github.

    cheat ls
# Displays everything in the target directory
ls path/to/the/target/directory

# Displays everything including hidden files
ls -a

# Displays all files, along with the size (with unit suffixes) and timestamp
ls -lh

# Display files, sorted by size
ls -S

# Display directories only
ls -d */

# Display directories only, include hidden
ls -d .*/ */

  3. manly - great for understanding command options without having to open manpage Github.

    $ manly ls -logX

ls - list directory contents
============================

  -g     like -l, but do not list owner

  -l     use a long listing format

  -o     like -l, but do not list group information

  -X     sort alphabetically by entry extension
